# Glossary

> Canonical terms used across this project. All agents (coding, review, etc.) should use these terms consistently.
>
> **Last updated**: YYYY-MM-DD

---

## Entries

- **Sprite**: A single image or frame that is part of a larger animation.
- **Sprite Sheet**: A single image file containing multiple smaller graphics (sprites) arranged in a grid.
- **Frame Strip**: A visual representation of all the individual frames extracted from the sprite sheet.
- **Offset (X/Y)**: The starting pixel coordinate on the sprite sheet from where the grid begins.
